##  开博缘由

*  最初的想法和动力就是将各种数学素材整合到云端，便于自己教学和学生学习，也便于别人使用。

*  记录自己的电脑学习心得，积累一些编程代码，满足自己的程序员之梦。

*  记录整理自己的数学专业的学习和教学感悟。

*  用思维导图将各种相关数学知识点给串接起来，便于理解学习。

*  将难以理解的内容作成动态的课件，直接嵌入到博客，下次使用非常方便，利人利己。
